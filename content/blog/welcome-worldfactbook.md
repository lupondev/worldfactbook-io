The CIA World Factbook shaped how millions of people read country-level intelligence. When public updates stopped, we wanted a place that keeps the same **structured profile idea** — geography, economy, society, government, military — with **live indicators** and an **open JSON API**.

## What you will find here

- **Country profiles** aligned with familiar Factbook-style sections.
- **Rankings and compare** tools for GDP, population, HDI, and more.
- **Weekly refreshes** from public sources (World Bank, IMF, UN, Freedom House, and others).

We publish short notes on data changes, methodology, and product updates on this blog.

## Try the API

```bash
curl -s "https://worldfactbook.io/api/v1/countries/" | head -c 400
```

No key required. Responses are JSON with CORS enabled for browser use.

> We do not replicate classified material. Everything here is built from **public-domain or openly licensed** datasets and the community Factbook JSON project.
